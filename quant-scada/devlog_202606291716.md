## Python 数据采集 (quant-scada-py)

### 架构
```
config/      — .env + settings.py
data/        — source.py (zzshare) / writer.py (TDengine) / stock_loader.py (MySQL)
core/        — collector.py (采集编排)
component/   — routes.py (Flask API)
scripts/     — sync_stocks.py (全量股票同步)
```

### 功能
- 启动时从 MySQL `stock_base_info` 加载全量 A 股代码 (5537 只)
- `zzshare.trade_days()` 判断交易日 + 9:30-11:30/13:00-15:00 时段判断
- 每 20s 分 20 批并行拉取实时行情 (6 线程 API + 8 线程 TDengine 写入)，1min=60次，留 40 次余量
- 每 5min 拉取市场情绪 → `intraday_sentiment` + `market_hist_sentiment`
- `sync_stocks.py` — zzshare `stock_basic()` 一次拉取全量 A 股，写入 MySQL stock_base_info
- taos-ws-py 适配：DSN 密码 `quote_plus` 编码，INSERT 用 `execute()` 而非 `query()`
- 数据库名改为下划线：`quant_scada_rt` / `quant_scada_hist`
